# 文件夹说明

- `raw_datasets`：原始数据集文件夹
- `processed`：处理后的数据集文件夹
- `results`：实验结果文件夹
- `train`：训练集文件夹
- `test`：测试集文件夹
- `Qwen`：存放大模型的文件夹（如果文件夹是空的，需要先运行`load_model.py`）

# Python文件说明

- `load_model.py`：用于加载模型和分词器
- `Prompt.py`：用于把数据集转换成提示词
- `preprocessing.py`：用于把原始数据集转换成处理后的数据集
- `system.py`：用于把提示词发给模型，获取模型回复
- `assessment.py`：用于评估模型回复的指标
- `main.py`：主文件

# 数据文件说明

## 原始数据集

原始数据集的特征名为英文，标签名是' Label'（前面有个空格），其中'BENIGN'表示正常流量，其他的属于恶意流量。

## 处理后的数据集

处理后的数据集样式：

- `instruction`：系统提示词（告诉模型是什么身份，要干什么）
- `input`：用户输入（用Prompt.py的函数生成）
- `output`：模型回复（正常流量/恶意流量）

## 结果文件

结果文件样式：

- `label`：样本实际标签，0->正常流量，1->恶意流量
- `response`：模型回复，0->正常流量，1->恶意流量
- `model_response`：模型回复的原始文本，用于检验模型回复是否合规
- `time`：模型预测花费的时间，单位为秒

## 日志文件

带log后缀的文件是日志文件，不用管
